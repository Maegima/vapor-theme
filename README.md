# Vapor Themes for VS Code

Welcome to the Vapor theme familyâ€”a set of themes designed to bring the atmospheric, electric feel of synthwave and outrun aesthetics directly into your Visual Studio Code environment.

This extension includes two distinct variations:

- **Vapor Pulse**: A high-contrast, vibrant theme that radiates energy.
- **Vapor Abyss**: A deep, muted theme with a moody, ambient vibe.

## Usage

Open the Command Palette (Ctrl+Shift+P or Cmd+Shift+P), type "Color Theme", and select either **Vapor Pulse** or **Vapor Abyss** from the list.
